# Responsive Sign-In Webpage

A clean, modern, and fully responsive Sign-In / Login webpage built using pure HTML5 and CSS3. This project is designed to be a lightweight starting template.

## Features

* *Pure HTML & CSS5:* No heavy frameworks or JavaScript dependencies—just lightweight, semantic code.
* *Modern UI/UX:* Features a minimalist aesthetic with smooth hover effects, styled input fields, and a subtle drop shadow card layout.
* *Accessible:* Built using semantic HTML tags (<form>, <input>, <label>) to ensure screen-reader friendliness.
